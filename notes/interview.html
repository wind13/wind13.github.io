面试题

Alan 测试题的最佳答案：

  1. 第一题完全实现难度有点大，建议分数权重占 70%；

  2. 第二题相对简单一点，建议分数权重占 30%；

电话面试题

  • 简介你的技术特长和优势
  • Vue.js 与 React.js 与 Angular.js 的区别和优劣；
  • 在 Angular 中写 directive 要传哪些参数？
  • React 的高阶组件 HOC 怎么用？
  • 对 Redux 中的 Reducer 的理解。
  • 前后端分离后的测试方法。
  • 英语读写听说能力如何？
  • Git 的分支怎么建立
  • 正则表达式了解吗？ ()? 是什么意思？
  • 跨域问题的解决方案有哪些？各有什么利弊？
  • 未来的职业规划，目前最想提高的技能？
  • 项目管理
    • 管理团队经验如何？
    • 项目代码的质量如何提高？ Code review 和 Unit test 等，XP 结对编程，类库、组件等。
    • 你对提高代码质量有什么心得体会。
  • 算法题
    • 一个人员列表数组，包含姓名和年龄，请说一下如何得到按年龄段分组的名单，如：90后:[张三，李四]。 答：for 循环
    • 假如有一个巨大的数组，如何用最有效的方法找到它的中值？不愿回答，直说不会。
  • 智力题：
    • 两部手机，100级台阶，最佳步骤测试多高摔坏。不愿回答，直说不会。
