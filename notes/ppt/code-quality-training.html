目前 Review 代码的几个问题

  • 使用 for 循环处理数组或对象；
  • 过多的 if ... else ... 判断语句；
  • 啰嗦的表达方法；

用 Lodash 处理数组和对象

  • lodash.map
  • lodash.each
  • lodash.pick
  • lodash.filter

在 BackBase 框架中 lpCoreUtils 封装了 lodash 库

使用 js 对象重构判断分支

  • 将代码中的 hard code 组织到常量中；
  • 将 switch 或 if 的判断分类组织到映射对象中；
  • 然后用映射对象直接获取对应的数值；

一些常用的简洁写法

  • if (utils.defined(x)) {...};
  • var x = condition1 && condition2;
  • if (!!defined) {...};
  • var x = objTime['H'];
