全栈开发人员

职位描述：

参与公司各个业务系统的软件开发，需要具备极强的学习能力和问题解决能力，具备独立工作的自我驱动力。

技术要求

  • 精通 C# 及 Windows 客户端软件开发相关技术栈，熟悉 Windows 打印控制相关开发，熟悉 Rust 语言优先；
  • 精通微信公众号、小程序开发，熟悉 uni-app，chameleon 或 taro 优先；
  • 精通 JavaScript，熟悉 Node，Vue，TypeScript 相关技术栈；
  • 熟悉 TDD 及 DevOps 相关技术栈，深入理解敏捷软件工程实践；
  • 熟悉 PHP 及 ThinkPHP 相关技术栈，熟悉 Linux 服务器管理；

以上技术至少精通掌握两个方面，多方面者优先，具备后期持续学习能力，快速学习相关技术并快速胜任相应软件开发工作者优先。
